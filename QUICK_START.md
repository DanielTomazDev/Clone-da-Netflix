# ⚡ Início Rápido

## 3 Passos para Começar

### 1️⃣ Instalar
```bash
npm install
```

### 2️⃣ Configurar API
Crie arquivo `.env` na raiz:
```
VITE_TMDB_API_KEY=sua_api_key_aqui
```

**Obter API Key**: https://www.themoviedb.org/settings/api

### 3️⃣ Rodar
```bash
npm run dev
```

Acesse: http://localhost:5173
